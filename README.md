# trivy-plugin-vuln-prio

[![License: Apache 2.0](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

Trivy output plugin that enriches vulnerability scan results with
EPSS exploit prediction scores, CISA KEV catalog data, and a composite
risk score for vulnerability prioritization.

> Work in progress
